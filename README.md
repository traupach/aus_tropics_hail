# Software for "Damaging surface hail in simulations of satellite-detected hailstorms in Australia's tropics"

This repository contains code used for the research study "Damaging surface hail in simulations of satellite-detected hailstorms in Australia's tropics" by T. H. Raupach and S. D. Bang.

## Required python packages:

* xarray_parcel by T. H. Raupach (https://doi.org/10.5281/zenodo.15081094) version >= v1.0.8.
